# Angel-in-Pocket / Ангел-в-кармане

**Билингвальный репозиторий** (Русский / English)

Персональный ИИ на основе GRA-обнуления с поддержкой мультивалютных подписок и локализации.

## Быстрый старт

```bash
pip install -r requirements.txt
python examples/run_angel.py
```

## Структура

- `angel/localization/` — языковые файлы (ru, en)
- `angel/subscription/` — управление подписками и платежами
- `angel/interface/telegram_bot.py` — билингвальный Telegram-бот
- `config/subscription_plans.yaml` — тарифы с мультивалютными ценами

## Подписка

Бот автоматически определяет язык пользователя и показывает цены в местной валюте:
- Русскоязычные пользователи — рубли (₽)
- Остальные — доллары ($) или евро (€)
